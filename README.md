# WaWa - PTAたすけあいアプリ

保護者と地域がつながるボランティア・ありがとうアプリのプロトタイプです。

## GitHub Pagesへのデプロイ手順

1. GitHubで新しいリポジトリを作成（例: `wawa-pta`）
2. このファイル群をアップロード
   ```
   index.html   ← メインアプリ
   manifest.json ← PWA設定
   README.md    ← このファイル
   ```
3. Settings → Pages → Source: `main` ブランチ、`/ (root)` を選択
4. Save → 数分後に `https://ユーザー名.github.io/wawa-pta` で公開

## PWA対応（スマホのホーム画面に追加）

iPhoneの場合:
- Safariで開く → 共有ボタン → 「ホーム画面に追加」

Androidの場合:
- Chromeで開く → メニュー → 「ホーム画面に追加」

## 機能一覧

- オンボーディング（ニックネーム・学年・メール登録）
- ホーム（実績・WaPTガイド・スタンプカード・参加履歴・通知）
- QR参加（スキャン→ポイント付与）
- 参加（ボランティア募集一覧・申込）
- ありがとう（学年フィルター・送信・フィード）
- クーポン（スタンプコンプリート→ノベルティ交換）
- 設定（学年変更・通知オン/オフ・退会）
- 管理者画面（集計・QR発行・ノベルティ設定）
